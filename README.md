# Поле чудес

Для запуска требуется python

#### Запуск сервера:
``` python server.py ```

(должен быть в папке с config и words.txt)

#### Запуск клиента:
``` python client.py ```

(по дефолту указан тот же ip и порт, что в конфиге. Можно раскомментировать код, тогда при запуске будет их спрашивать)
